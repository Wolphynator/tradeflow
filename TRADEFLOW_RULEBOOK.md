## 1. Status / Version

- **Status:** Draft

## 2. Purpose

This document is a safe placeholder for the future single source of truth for TradeFlow Pro rules, business logic, security decisions, UI principles, permanent decisions, and future ideas.

It will eventually collect reviewed and approved project decisions in one place. For now, it defines no new app behaviour and must not be used to replace existing approved project documentation or instructions.

## 3. Current Warning

> **This document is not fully reviewed yet. Do not treat rules inside this file as final until marked Approved.**

Before any future AI assistant implements or changes rules from this file, it must ask the Project Owner for confirmation if the relevant section is still marked **Draft**.
